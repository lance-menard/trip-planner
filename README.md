# Trip Planner

## Setup

* Update example.env with your API key
* Rename the file to .env
* Run `npm run go` to do the following:
  * `npm i`
  * `npm run bootstrap`
  * `npm run image`
  * `docker-compose up -d`
  * `start localhost:8080`
